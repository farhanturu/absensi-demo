## Features

- **Login** — Authentication dengan JWT simulation
- **Dashboard** — Statistik, status hari ini, riwayat terakhir
- **Check-in/Check-out** — Jam real-time, satu klik absen
- **Riwayat** — Kalender mingguan dengan warna status
- **Cuti** — Pengajuan cuti dengan status approval
- **Laporan** — Rekap absensi per tanggal (admin)
- **Karyawan** — CRUD karyawan (admin)
- **Responsive** — Mobile & desktop
- **Dark theme** — Modern UI dengan gradient

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- localStorage sebagai database

## How to Use

1. Buka demo link
2. Login dengan akun demo
3. Coba fitur absen, riwayat, cuti
4. Login sebagai admin untuk fitur laporan & karyawan
